# 🤖 Bidu AI

Bidu AI is a smart AI-powered personal assistant designed to enhance productivity through intelligent conversations, document processing, image understanding, and AI-driven assistance.

The application provides users with a simple and interactive interface to access advanced AI capabilities in one place.

---

## 🚀 Features

- AI-powered conversational assistant
- Document analysis and summarization
- Image understanding and interpretation
- Question answering from uploaded files
- Smart information retrieval
- Interactive Streamlit interface
- Fast and user-friendly experience

---

## 🛠️ Technologies Used

- Python
- Streamlit
- OpenAI APIs
- LangChain
- Pandas
- AI/ML Libraries

## ⚙️ How It Works

1. User enters a query.
2. Bidu AI processes the request.
3. Relevant AI models analyze the input.
4. The system generates intelligent responses.
5. Results are displayed through an interactive interface.
